# V2exClient

V2exClient, 基于MVP，以Dagger2 + RxJava2 + Retrofit + Butterknife + RxBus打造的Android基础框架，使用Material Design设计风格，使用Glide加载图片，以V2EX-API为数据来源的一款 V2ex简易客户端。
